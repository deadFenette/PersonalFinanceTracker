# Personal Finance Tracker

Простое приложение для учета личных финансов, разработанное на .NET и WPF.

## Описание

Это десктопное приложение помогает отслеживать личные финансы через систему кошельков и транзакций. Позволяет вести учет доходов и расходов, анализировать траты и контролировать балансы.

## Основные возможности

- **Управление кошельками** - создание кошельков с разными валютами
- **Учет транзакций** - добавление доходов и расходов с описанием
- **Контроль баланса** - автоматическая проверка достаточности средств
- **Аналитика** - группировка транзакций по типам и периодам
- **Топ-траты** - отображение самых крупных расходов по кошелькам
- **Сохранение данных** - работа с JSON файлами для хранения информации

## Технологии

- .NET 8.0
- WPF (Windows Presentation Foundation)
- C# 
- JSON для хранения данных
- xUnit для тестирования

## Требования

- Windows 10/11
- .NET 8.0 Runtime
- 100 МБ свободного мест

Запуск из Visual Studio:
1. Открыть `PersonalFinanceTracker.sln`.
2. Собрать конфигурацию `Debug` или `Release`.
3. Запустить проект `PersonalFinanceTracker`.

Запуск из CLI:
```powershell
dotnet build
dotnet run --project PersonalFinanceTracker.csproj
```

При первом старте приложение предложит сгенерировать тестовые данные, если сохранённого файла нет.

## Текущие ограничения
- Нет нормализации валют: суммы из разных кошельков агрегируются без конвертации, что искажает общую картину при смешанных валютах.
- В интерфейсе группировка транзакций отображается линейным списком без заголовков групп; визуально различить типы можно только по колонкам.
- Валидация пользовательского ввода минимальна (нет проверки диапазонов дат, типов валют и т.п.).
- Логи сервиса файлов выводятся в консоль, что неудобно в WPF-контексте.

## Тестирование
Проект включает unit-тесты, покрывающие основную бизнес-логику. Для запуска тестов используйте Test Explorer в Visual Studio.
